This file contains a description of the tools available in the window Calibration Tool.

1 Main menu (on the left)

1.1 Options Minerals' phases

Select the mineral phase in the right menu. The correpondent map (what map is this one??) is displayed in the bottom right. The graphic in the central part exhibts the calibration curves for all the elements, whose parameters are available in the table below.

To display the calibration curve of each element, click on the arrow on the left side of the phases' name opening a dropdown menu with the elements' list.

By selecting an element, its calibration curve is displayed as well as the correspondent quantitative map (oxide wt-%).

When an element is selected the button Adjust appears above the main menu.

1.1.1 Button Adjust

By clicking on this button the values filled in the fields Slope and Background can be adjusted, by tapping the new values.

Press enter to visualize the alterations on the graphic.

1.2 Option General fit

By selecting the General fit on the menu positioned on the left, a graphic with the calibration curves for all the elements are displayed.

Note: The red spots are not considered in the calculation of the calibration curves'.

Button Apply standardization

Click on this button to calibrate the map.

Note: No adjustment on the calibrations' curves in the general fit is possible. The calibration is authomatically generated by the program.

By applying the standardization, the option Quanti in the main menu becomes available. There the quantitative maps in oxide wt-% of each phase can be displayed.

The option Merged in the main menu also becomes available after the standardization. The quantified merged maps (i.e., quantitative maps in oxide wt-% for all the phases together) are available for each oxide.

Note: By hoving the cursor on the images an image menu will appear on the upper right. This menu includes options to zoom, save and copy the images.
