This file contains a description of the tools available in the main tab Calibrate.

The following steps are required to convert X-ray semi-quantitative maps and trace elements maps into fully quantitative maps of chemical composition.

1. EPMA Standard Data

The standards constitute WDS spot analyses to be used as internal standards that will define the calibration curve that correlates X-ray intensity and oxide wt-% composition.

Use those tools to import the standard file, add extra standards, apply stardards' position corrections and check the chemical composition of each spot analyses into XMapTools.

1.1 Button Import (Import spots for Standards (from file))

Select the standards from the file 'standards.txt' if the box immediately below Import from Standards.txt is selected (default).

If the file has a different name, unselect the box and select the file in the Pick a file pop-up window.

The spot standards will be loaded in the displayer and each spot can be selected in the Secondary Menu Standards(Spots) on the left bar.

1.2 Button Std Mod (Open Standard Data Viewer)

To activate this button, the function Standards (Spots) must be selected in the Secondary Menu (on the left).

With this function it is possible to see a table with the composition (intensity and oxide-wt%) of each standard, check the quality and adjust the standards/maps positions and add new individual standards (together with the button Add standard point described below.

By clicking on this button the right side of the window is activated.

1.2.1 Tab Data

Use this option to visualize the element's composition of the standards in X-ray intensity in the 'Int' column and the correspondent oxide composition (wt%) in the column 'wt%'.

A standard must be selected in the function Standards (Spots) in the main menu (on the left).

1.2.2 Tab Plot

Functions available in this tab allows the user to check the quality and correct the position of the standards.

To change the displayed element, close the window Data Standard Viewer, select another element on Intensity and click again in the button Std Mod to reopen the window.

1.2.2.1 Plot intensity and the composition (wt%) versus standards' positions

The graphic in the upper part of the window displays the compositions of the internal standard in wt% (red, right Y axis) and their respective intensities of the corresponding pixels (blue, left Y axis) for the spot analysis' position.

1.2.2.2 Boxes Check quality of standards/maps positions

This function automatically searches the optimal positions of the spot analyses (internal standard) on the maps.

In the box on top, a correlation between the compositions of the spot analyses (in oxide wt-%) and the compositions of the corresponding pixels (in counts) is calculated and the optimal position for this correlation is show in the center, in yellow.

The bottom box displays the optimal position (higher number) for all the elements are calculated as the sum of square of correlation coefficients of each element.

The blue circle represents the current correlation. If it is not positioned in the optimal position (yellow in the colorbar positioned on the right) it means there might have been some displacement between the maps and the spot analyses.

To adjust the position of the standards, hover the cursor over the blue circle showing up a transparent circle. Click on it and move the blue circle to the new position. The values in the two white field on top will be adjusted. Then click on the button Refresh to update the position in the graphic and in the boxes.

Note: To activate the Refresh button, the option Standards (Spots) in the main menu (on the left) must be selected.

Note: If no good correlation exists, the higher value of the second figure could not be the optimal position. In this case a problem should have occurred with the transformation of the map coordinates or the best position can be out of range.

1.3 Button Add standard point

To activate this function, the button Std Mod (on the left) must be selected.

Then, by pressing the button Add Standard Point, a cursor is activated.

The new standard will be added by right clicking in the selected position.

It is posible to adjust the position of the new standard by moving the blue spot (transparent circle) or its label (cross-shaped cursor).

Select the new standards on the Secondary Menu Standards(Calibration) on the left bar and the correpondent informtion table will become available in the Data tab located in the Standard Data Viewer on the right side of the window.

Add the correspondent oxide (wt%) for the new standard in the column 'wt%' in blue. The correspondent X-ray intensity is already available in 'Int' column.

???? To save the new data, press the exit button 'X' on the top right to close the window.

To rename the spot, select and left click on its name located in the Standards(Spots) on the Secondary Menu.

To activate this button, a mask file must be selected in the function Mask File on the Secondary Menu (on the left).

By clicking on this button the Calibration Tool window pops up.

3. Local Bulk Composition Extractor

As discussed in Lanari and Engi (2017), it is necessary to apply a density correction prior exporting any local bulk composition.

3.1 Button Density (Generate Density Map (from a mask file))

Select a mask file to activate this button.

By clicking on this button, the window ??? opens.

The average density of each mineral phase must be filled in the window. Press enter to save the values.

Note: Wether the full names of minerals in the classification are employed, the density values will be authomatically taken from XMapTools' database.

Note: The density values can be obtained from the website webmineral.com

A Density map will be created and stored in the option Other in the Primary Menu under the name 'Density [1_RandomForest]' (in brackets the name of the algorithm will be shown).

3.2 Button Merge (Merge Quanti Data)

Click on this button merge quantitative maps.

To activate this button, select a phase map on the Primary Menu Quanti.

In the window that pop-ups, select the phases' maps to be merged. Click on 'Ok' to generated the merged maps.

3.3 Button LBC Mod (Open Bulk Composition Viewer)

To activate this button, a merged map must be selected on the Primary Menu on the left side Merged.

??? Click on this button to activate the Display module on the right side of the window and activate the button Add a ROI for Local Bulk extraction.

The main goal of this option is to extract local bulk composition (LBC) for thermodynamic modeling and pseudomorphic reconstructions.

3.3.1 Top window buttons

3.3.1.1 Dropdown menu

Use this menu to select the density map to be used in the calculation of the Bulk Composition.

3.3.1.2 Button Camera Copy Data to clipboard

Use this button to copy the data from the table and paste in a Excel spreadsheet, for instance.

3.3.1.3 Button Diskette Save

Use this button to save the data in the table as a .txt file.

3.3.2. Tab Data

After the window pops-up, press the button Add a ROI for Local Bulk extraction to select the local domain from which the local bulk must be extracted.

When an area is selected, the table LBC (local bulk composition) extractor in the window Bulk Composition Viewer displays the composition in oxide (wt%) of the domain. Below the table, a pie chart shows the numerical proportion of the oxides.

The selected area can be changed with no need to close this window. The values in the table are authomatically updated.

5. LA-ICP-MS Tools

Use those tools to convert intensity maps into quantitative maps (% or ppm).

5.1 Button Import (Import Maps for Standards (from file))

By pressing this button, a window pops-up. Select and open the file 'MapStandards_Import'.

This steps will open the standards' files that become available in the option Standards (Maps) in the Secondary Menu.

5.2 Button Calibrate (Open LA-ICP-MS Calibration Tools)

To apply the calibration, a maksfile must be selected on the Mask file option in the Secondary Menu.

By pressing this button the window 'Data Calibration LA-ICP-MS - XMapTools' pops-up.

5.2.1 Window Data Calibration LA-ICP-MS - XMapTools

5.2.1.1 Dropdown menu Mineral

Select the mineral phase to be used as internal standard.

5.2.1.2 Dropdown menu Internal Standard

Select the element to be used as internal standard.

5.2.1.3 Boxes Fixed composition and Variable composition

Depending on wether the mineral is zoned or not, select one of the boxes.

5.2.1.4 Internal Composition Converter

Use this function to convert the composition in oxide % from your sample to be used as internal standard in the upper part of the window.

5.2.1.4.1 Dropdown menu (wt% Oxide)

Use this menu to select the oxide to be converted.

Enter the value in wt% Oxide in the empty box on the right side of the dropdown menu and press enter. The converted value to element in ug/g will be exhibited on the box on its right.

Copy the converted value and past it on the empty box on the upper part of the window below the dropdown menu Element.

By doing this, two plots are displayed on the bottom of the window. On the left side, the 'k matrix', and on the right side, the calibrated map in ppm.

Press the button Apply to generate the quantitative maps that become available in the option Quanti on the Primary Menu. The maps are in ppm. To convert them to wt% Oxide, right click on a map and select the option Convert. A window with the conversion methods pops-up.

Note: After converting the maps, the name of the maps must be renamed in the Primary Menu.

    Variable composition

Note: Wether the mineral phase is zoned, the Variable composition must be selected.

By selecting this option a table becomes available in the main part of the window and just above a button with a target.

Press the button to activate a circle and position it in the desired zone in the 'Intensity map (in cps)'. Enter the wt% Oxide composition and click on the table in the correspondent field 'Intensity (cps)' and after in the 'Composition (ug/g), the converted values will be authomatically read.

Repeat this step for the following zones.

Note: By pressing Apply the generated maps will have the same name and should be renamed in Quanti on the Primary Menu.

5.3 Button Spider (Open Spider Module)

By pressing this button, the window 'Spider Module' pops-up.

5.3.1 Dropdown menu Normalization

Select the work to be used in the normalization.

5.3.2 Dropdown menu Elements

Select the element's map to de displayed. Use the button Magic Wand on its right side to adjust the constrast.

5.3.2 Sampling

The compositions to be plotted on the right side of the maps can be delimited by selecting one of the options 'Circle', 'ROI' or 'Path'.

Note: Click on the box Hold on to select multiple areas. It is posible to edit and move the positions of the selected areas and the plot changes live.

Note: The uncertainties are plotted on the diagram. It is possible to select what type of uncertainty '1se' or '1sd', should be displayed on the dropdown menu Uncertainty.
