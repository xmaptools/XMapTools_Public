Detailed description


This help file contains a description of the Converter for EPMA data which can natively read data from JEOL and CAMECA microprobes.

Main steps

  • Select the data format (e.g. JEOL or CAMECA)
  • Select the destination folder, usually an empty folder where data with a compatible format will be saved
  • Select maps in a given folder; text or csv map files generated by the microprobe software
  • Validate map selection
  • Add standard data exported as text or csv files from the microprobe software (can be repeated until all standard data has been imported)
  • Generate Standards.txt containing the map coordinates and the spot analysis data to be used as internal standards for map calibration.

Detailed description

Use the Select the format drop down menu to select data format.

The following options are available:
  • EPMA - JEOL (SUN) for old JEOL instruments
  • EPMA - JEOL (WINDOWS) for recent JEOL instruments
  • EPMA - CAMECA for all CAMECA instruments

After selecting the data format, two text boxes are displayed indicating the types of files required for the selected format. The first on the left is for the map and the second for the spot analyses.

  • JEOL (SUN) requires two files for each map: 'XX.cnd' and 'XX_map.txt', and two files for each spot analysis dataset: 'summary.txt' and 'stage.txt'
  • JEOL (WINDOWS) requires two files for each map: 'data00X.cnd' and 'data00X.csv', and a single file for each spot analysis dataset: 'summary.csv'
  • CAMECA requires a single file for each map: 'XXXX.txt', and a single file for each spot analysis dataset: 'YYY.csv'

After selecting the data format, press the Set a destination folder button and select an empty folder to save the data in a compatible format. This folder must be empty and existing data will be deleted!

After selecting the destination folder, press the Import Maps button to select the folder containing the map files. XMapTools will convert all maps available in this folder.

Once the maps have been selected, press the Validate map selection button to continue.

Select the Add Standard Data button to add a set of spot analyses and select the appropriate file(s). In the case of JEOL (SUN) two files must be selected, first the file containing the mass fractions (usually summary.txt) and immediately after that the file containing the stage coordinates for the same points (stage.txt).

Once all the files for the spot analyses have been selected, press the Generate Standards.txt button. This will complete the procedure and close the converter. Note that when you return to XMapTools, the working directory has been adjusted and you can start importing your maps.
