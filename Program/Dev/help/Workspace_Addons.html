Bingo-Antidote

A demo version of Bingo-Antidote is included in this version of XMapTools. Bingo-Antidote is a program for iterative thermodynamic modeling and it requires Theriak-Domino for Gibbs free energy minimisation.

Detailed instruction on how to use Bingo-Antidote will be provided in a future release.
