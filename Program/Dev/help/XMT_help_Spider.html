This help file contains a description of the Spider module which can be used to generate spider plots.

Step 1 select the normalisation data using the dropdown menu Normalization

Step 2 (optional) use the button Auto contrast to automatically adjust the color range.

Step 3 pick pixels to plot using one of the ROI type: Circle , ROI or Path .

Step 4 (optional) activate the mode Hold on to select multiple region-of-interest. Simply repeat step 3 to add more selections (circle or ROI). It is possible to edit and move tany selection and the plot will be updated.

Note: Uncertainties are plotted on the diagram. It is possible to select if you want to display '1se' (1sd/sqrt(N)) or '1sd' .
